About This Project

This dashboard visualizes real-time data from weather balloons, integrating meteorological data with a LSTM machine learning model to predict balloon positions for the next hour. The balloon data updates every 2.5 minutes, while weather feeds update less frequently.

Data Sources:

  • Balloon Data: WindBorne API
  • Weather Data: Open-Meteo API
